# Screenshots

This folder contains screenshots and visual documentation for the Attendance Management System backend.

## Contents

- **output.png** - Sample output/demonstration of the backend application

## Purpose

These screenshots serve as:
- Visual documentation of the application's functionality
- Reference for expected behavior and output
- Examples for testing and validation purposes

## Backend Information

- **Application**: Attendance Management System
- **Framework**: Spring Boot 4.1.0-M1
- **Port**: 8080
- **Database**: H2 (in-memory)

## API Endpoints

The backend provides the following REST endpoints:

- `GET /attendance` - Retrieve all attendance records
- `POST /attendance` - Create a new attendance record
- `PUT /attendance/{id}` - Update an existing record
- `DELETE /attendance/{id}` - Delete a record
- `GET /h2-console` - Access H2 database console
